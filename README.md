# 0DaySiege

A tower defense game where players defend a network's Firewall against waves of malicious software.

## Platforms

- iOS
- Android
- macOS
- Windows

## Tech Stack

- Unity 6 (6000.2.7f2)
- Universal Render Pipeline (URP) with 2D renderer

- [Game Design Document](GDD.md)
- [Implementation Plan](PLAN.md)
